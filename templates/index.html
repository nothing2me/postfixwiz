What is Postfix Notation?

Postfix notation (also called Reverse Polish Notation or RPN) is a mathematical notation where operators follow their operands. Instead of writing 3 + 4, you write 3 4 +.

Example: Converting Infix to Postfix

Infix Expression:

a + b * c

Postfix Expression:

a b c * +

How it works:

  1. Operands go directly to output: a, b, c
  2. Operators are pushed to a stack based on precedence
  3. Higher precedence operators (*) are processed before lower precedence ones (+)
  4. Result: a b c * + (multiply first, then add)
